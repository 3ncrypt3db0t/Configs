## ❓ How To
- Install ZSH 
```
*************** APT Based Distros ***************
$~ sudo apt install zsh 

*************** RPM Based Distros ***************
$~ sudo yum install zsh 
$~ sudo dnf install zsh 

*************** Arch Based Distros ***************
$~ sudo pacman -S zsh 
```

- Using ZSH Shell (Optional)
```
I'm using XFCE4-Terminal
Go to Terminal's Prefrence -> General -> Check Working Directory -> /bin/zsh
```
